# news-parser
Парсер новостей из Telegram для генерации аналитических обзоров
## Итоги дня (24 июня 2026)

- Парсер настроен и работает в GitHub Actions.
- Сбор новостей автоматический, 4 раза в сутки (можно увеличить).
- База данных `news.db` кэшируется между запусками.
- Дубли исключены, накопление идёт.
- Ключи API хранятся в секретах.
- Следующий шаг: скрипт для генерации обзоров (`make_review.py`).
